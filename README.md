# JT808车载定位系统（带WEB、带数据库）

![](https://www.lingx.com/images/QQ20210420155620.png)
## 项目介绍
 - 后台基于Netty开发，支持分包，同时支持JT808-2013，JT808-2019
 - WEB采用灵犀模驱低代码框架
 - 数据库采用MySQL8.0
 - 开发环境：JDK1.8、Eclipse、tomcat8.0、MYSQL8.8
## 项目特色
 - 完整的前后端，可直接部署使用
 - 兼容2011、2013、2019部标协议版本，支持分包，支持版本标识；
 - 支持指令下发与参数设置
